Comprehensive Commands

Music Commands

/play [song] - Plays a song.

/skip - Skips the current song.

/queue - Shows the song queue.

/pause - Pauses the music.

/resume - Resumes the music.

/volume [level] - Sets the volume.

/loop - Loops the current track or queue.

/nowplaying - Shows the currently playing song.

/shuffle - Shuffles the queue.

AI & TTS Commands

/setup-ai - Sets up the AI features.

/toggle-tts - Toggles Text-to-Speech.

/select-tts-voice - Selects a TTS voice.

/setup-ai-key - Set your OpenAI API key.

Leveling Commands

/level - Shows your current level.

/leaderboard - Shows the server leaderboard.

/customize-level-card - Customizes your level card.

/setup-leveling - Enables the leveling system.

Utility Commands

/help - Displays all available commands.

/ping - Checks the bot's latency.

/support - Get links for support.

/clear - Bulk delete messages.
